# Quality Gate Checklist

## Must Pass (article rejected if any fail)
- [ ] Factual claims are sourced with links
- [ ] Unique angle - not just summarizing what others wrote
- [ ] Has actionable takeaway the reader can use today
- [ ] Working code example included and tested
- [ ] No fluff paragraphs (every paragraph adds value)
- [ ] Under 1200 words
- [ ] Title under 60 characters, SEO-friendly
- [ ] TL;DR is exactly 3 sentences

## Bonus (improves ranking)
- [ ] Includes Mermaid diagram
- [ ] References primary source (paper, repo, docs)
- [ ] Compares with alternatives
- [ ] Includes performance/benchmark data
